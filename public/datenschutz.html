Erhobene Daten

Bei der Nutzung dieser Website können folgende Daten erhoben werden:

Server-Logdaten

Beim Besuch der Website werden automatisch folgende technische Informationen erfasst:

  • IP-Adresse
  • Datum und Uhrzeit des Zugriffs
  • Aufgerufene Seiten
  • Browser-Typ und Version
  • Betriebssystem
  • Referrer-URL (zuvor besuchte Seite)

Diese Daten werden zur Gewährleistung der Systemsicherheit, zur Fehlerdiagnose und zur Analyse der Website-Nutzung erhoben. Eine Zusammenführung mit anderen Datenquellen erfolgt nicht.

Kontaktformular

Bei Nutzung des Kontaktformulars werden folgende Daten erhoben:

  • Name
  • E-Mail-Adresse
  • Nachricht
  • Ausgewählte Dienstleistung (optional)

Diese Daten werden ausschliesslich zur Bearbeitung Ihrer Anfrage und zur Kontaktaufnahme verwendet. Eine Weitergabe an Dritte erfolgt nicht, ausser zur technischen Zustellung der Nachricht (siehe Drittanbieter).

Zweck der Datenverarbeitung

  • Bereitstellung und Betrieb der Website
  • Bearbeitung von Kontaktanfragen
  • Projektabwicklung und Kommunikation mit Kunden
  • Technische Sicherheit und Fehleranalyse

Drittanbieter

Folgende Drittanbieter werden eingesetzt:

Cloudflare Pages (Hosting):
Diese Website wird über Cloudflare Pages gehostet. Cloudflare kann dabei technische Daten wie IP-Adressen verarbeiten. Weitere Informationen finden Sie in der Datenschutzerklärung von Cloudflare.

Resend (E-Mail-Versand):
Kontaktformular-Anfragen werden über Resend versendet. Resend verarbeitet dabei die übermittelten Daten (Name, E-Mail, Nachricht). Weitere Informationen: Resend Privacy Policy.

Google Fonts:
Diese Website nutzt Google Fonts zur Darstellung von Schriften. Beim Laden der Seite werden Schriftdateien von Google-Servern geladen, wodurch Ihre IP-Adresse an Google übermittelt wird. Weitere Informationen: Google Privacy Policy.

Cookies

Diese Website verwendet keine Tracking-Cookies oder Analyse-Tools. Es werden nur technisch notwendige Session-Cookies verwendet, die nach dem Schliessen des Browsers automatisch gelöscht werden.

Speicherdauer
